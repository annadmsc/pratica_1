Praticando com o Git e Github

O que é o GIT?

O Git é uma plataforma que permite salvar as diferentes alterações de um código/programa e manter um histórico para a melhor organização da administração.

O que é o Github

O Github é uma ramificação do Git, porém o Github segue o padrão de rede social e assim dando a opção de publicar seus repositórios para outros verem, e também é possível que empresas vejam seu trabalho por lá.
